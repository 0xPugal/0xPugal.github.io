Project #2

Leet Recon

LeetRecon stands out as a comprehensive web application reconnaissance framework, meticulously crafted to automate key aspects of security testing essential for bug bounties. Its versatile functionalities include Subdomain enumeration, CIDR mapping, Fuzzing, Directory Brute-forcing, Port scanning, origin IP identification, WAF bypass, Vulnerability scanning, JavaScript crawling, and more.

Python, Bash, Docker
